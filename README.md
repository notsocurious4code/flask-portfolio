**# Flask Portfolio Website**



\## Description

This project is a personal portfolio website built using Flask.  

It includes multiple sections to showcase skills and projects.



\## Features

\- About section

\- Skills grid

\- Projects section

\- Contact form

\- Dark mode toggle



\## Technologies Used

\- Python

\- Flask

\- HTML

\- CSS

\- JavaScript



\## How to Run the Project

1\. Install Flask:

&nbsp;  pip install flask



2\. Run the application:

&nbsp;  python app.py



3\. Open in browser:

&nbsp;  http://127.0.0.1:5000/
